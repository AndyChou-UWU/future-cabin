智慧小屋專案網站介紹
==================

網站名稱：FUTURE CABIN | 智慧小屋專案
網站網址：http://localhost:8000 (本地開發環境)
         https://fluviomarine-nonexpiatory-harlow.ngrok-free.dev/(全域網址)

專案概述
--------
這是一個基於Django框架開發的智慧小屋專案展示網站，整合了AI導覽、即時電路模擬與訪客回饋功能。網站旨在讓參觀者一眼看懂小屋功能與專案亮點，同時提供互動式體驗。

主要功能
--------
1. AI智慧導覽
   - 整合Ollama AI模型，提供即時問答功能
   - 支援語音朗讀與語音設定調整
   - 動態文字顯示效果

2. 電路設計圖預覽
   - 互動式設計圖展示
   - 支援縮放與拖拽操作
   - 鼠標懸停即可移動圖片

3. 訪客留言板
   - 即時留言提交與顯示
   - 留言列表可滾動瀏覽
   - 支援匿名留言

4. 系統狀態檢查
   - 檢查AI服務與網路連線狀態
   - 必須通過檢查才能進入聊天系統
   - 提供即時狀態回饋

5. 專案詳情頁面
   - 展示專案里程碑進度
   - 硬體清單與工作分配圖
   - 團隊成員介紹

技術架構
--------
- 後端框架：Django 4.x
- 前端技術：HTML5, CSS3, JavaScript (ES6+)
- AI整合：Ollama API
- 資料庫：SQLite3
- 樣式設計：自訂CSS，支援響應式佈局
- 語音功能：Web Speech API

專案結構
--------
smart_cabin_project/
├── my_cabin/                 # Django主應用
│   ├── models.py            # 資料模型
│   ├── views.py             # 視圖函數
│   ├── urls.py              # URL路由
│   └── settings.py          # 設定檔案
├── chatbot/                 # 聊天機器人應用
├── static/                  # 靜態檔案
├── templates/               # HTML模板
│   ├── chat.html            # 主聊天頁面
│   └── check.html           # 系統檢查頁面
├── manage.py                # Django管理腳本
├── db.sqlite3               # 資料庫檔案
└── comments.txt             # 訪客留言儲存
